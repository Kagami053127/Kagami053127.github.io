「HANA（哈娜酱）」配布页面

这里是伪春菜人格 hana（哈娜酱）的配布页。

# 人格介绍

基于 [小花仙] 页游世界观的原创人格。

又是一个在花园种粉色丁香，妄想发家致富的日子。
唯一不同的，今天在花园遇到的是……

  • 可爱 / 单纯 / 治愈 / 偶尔好笑 / 无好感度 / 无里模式 / 服装占卜 / 塔罗牌
  • 单纯希望她能陪伴你度过在拉贝尔大陆的美好时光
  • 支持进行关于服装搭配的撞色 / 风格 / 纯色 / 元素 / 部件占卜
  • 雷点：部分对话带有主观想象或意见 / 暗恋塔巴斯

下载地址：戳我

强烈建议您安装完毕后首先进行线上更新。（右键→项圈魔法）

# 食用说明

  • 第一次接触伪春菜，不会安装：戳我
  • 请尊重伪春菜（伺か）著作权质问中提到的内容。
  • 有任何意见、建议、BUG 反馈，欢迎留言：戳我

# 常见问题

  • 为什么我安装了，人格没有出现？
    安装完毕后，右键→切换人格→Hana
  • 为什么我的对话框和截图上的不一样？
    预览图的对话框为 BK 小姐制作的 Bubble。
    下载该对话框，拖到 Hana 的身上。
    然后右键→更改便签→选择 Bubble 即可。
  • 为什么我没有 [聊聊] 功能？
    请先阅读三封神秘信件。
  • 游戏服装每周都会更新，可以占卜到最近服装吗？
    作者会定期更新占卜模块的元素内容，请经常更新。
    更新方式：右键人格→项圈魔法。
  • [聊聊] 没有反应？
    她目前能听懂的话语还不是很多。
  • 如何修改她对我的称呼，还有我的出生日期？
    双击 Hana→设定→我的资料。
  • 想让她时刻保持在屏幕最上面？
    双击 Hana→设定→保持上层。
  • 封面图好看，可以拿走做头像吗？
    是我的约稿（虽然是草图版，不可以的 Q Q
