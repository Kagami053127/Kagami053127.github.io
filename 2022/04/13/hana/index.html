这里是伪春菜人格 Hana（哈娜酱）的配布页。

# 人格介绍

使用瞬祭蓝果 的『Nothing in the shell』范例人格修改而来。
GHOST：ShawRoot
FREE SHELL：Nollety

基于 [小花仙] 页游世界观的原创人格。

又是一个在花园种粉色丁香，妄想发家致富的日子。
唯一不同的，今天在花园遇到的是……

  • 可爱 / 单纯 / 治愈 / 偶尔好笑 / 单纯希望她能陪伴你度过在拉贝尔大陆的美好时光
  • 无好感度 / 无里模式 / 服装占卜 / 塔罗牌 / 每日运势 / 成就收集
  • 支持进行关于服装搭配的撞色 / 风格 / 纯色 / 元素 / 部件占卜
  • 雷点：部分对话带有主观的想象或吐槽 / 暗恋塔巴斯

下载地址：戳我

# 常见问题

  • 为什么我安装了，人格没有出现？
    安装完毕后，右键→切换人格→Hana
  • 为什么我没有 [聊聊] 功能？
    请先阅读三封神秘信件。
  • 游戏服装每周都会更新，可以占卜到最近服装吗？
    作者会定期更新 [穿搭占卜] 模块的内容，请经常更新。
    更新方式：右键人格→项圈魔法。
  • [聊聊] 没有反应？
    她目前能听懂的话语还不是很多。
  • 如何修改她对我的称呼，还有我的出生日期？
    双击 Hana→设定→我的资料。
  • 想让她时刻保持在屏幕最上面？
    双击 Hana→设定→保持上层。
    或 右键→常用选项→保持显示在最上层。
  • 封面图可以拿走做头像吗？
    是我的约稿 不可以的 Q Q
