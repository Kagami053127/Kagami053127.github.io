# 什么是伪春菜

伪春菜又称为伺か，简单的来说就是一个桌面宠物。开发的目的就是能为用户处理电脑大大小小的事情，包括报时、检查邮件、聊天、对时、检视 CPU 的使用度、关心主人的身体等等。
伪春菜的构造很简单，分别由三个部分所组成：核心程式（本体）、人格（ghost，ゴースト）和外壳（shell，シェル)。

名称 功能
核心程式（本体） 以一台电脑来比喻，本体就如同操作系统（如 Windows），伪春菜的本体叫 SSP。
人格（ghost） 指的就是人工 AI 的部分，包含了许多的程式码、和用户的对话语句、时间指令等。
外壳（shell） 指的就是我们在屏幕上看到的人物外型。

伪春菜实际上可用的功能，随人格（ghost）的不同而不同。

我们所作的人格（ghost）都是基于核心程式 SSP 上的，就像软件要基于操作系统一样。

# 第一步：核心程式（SSP）的下载

这里提供一个下载页面：戳我，在这里选择一个版本的 SSP 下载即可。
建议下载 CSSP (简体版 SSP) 2.02.90。
另外，如果下载后，提示需要更新，不建议更新，否则软件界面可能会变成日文（。
下载完毕后，双击 ssp.exe（蝴蝶结图标），屏幕右下角出现一个绿色的，在说话的桌面宠物，即代表安装成功。

# 第二步：人格（ghost）的安装

  • 双击 ssp.exe，打开上一步安装的 SSP。
  • 把以 nar 结尾的人格安装包文件拖到默认的绿色桌面宠物身上即可安装成功。
  • 如果不幸提示安装失败了，请看第三步。

# 第三步：人格（ghost）安装失败的原因

可能有以下几种原因 : (

  • 人格文件未下载完成就拖到 SSP 中安装。
  • 人格文件损坏。
  • 遇到这种问题可以联系我。
