# 伪春菜（伺か）著作权质问

此页适用于本站所有配布物。
在下载 / 安装人格之前请务必完成以下内容的阅读 ^^

1. 直接链接到人格公开网站的首页
可以。
2. 对人格的内容进行剧透 / 捏他 (ネタバレ)
不是很建议。
3. 登录到搜索引擎上
没问题。
4. 直接链接到人格的下载页面（此处指配布页面，即包含人格说明等的完整页面）
欢迎 ^^！
5. 写感想或 repo 文
非常欢迎！换句话说，我会感到非常荣幸。
6. 报告故障或错字
感谢您的报告。
7. 提出「希望追加 XX」之类的愿望
可以，在我能力范围内的话。
8. 使用有关人格的捏他
请和我联系。
9. 写出批评性的意见
如果掌握好度不是挑刺来的话。
10. 偷看辞书文件
不是很建议，我会很不舒服。
11. 修改存档文件
本地的话随便。请不要分享到互联网。
12. 进行二次创作 (绘画、小说、漫画、游戏等)
非常欢迎！
14. 进行和原作的设定离得很远的二次创作
随便啦。
15. 进行带有暴力、猎奇元素的二次创作
不介意。
16. 进行含有性描写的 18 禁二次创作
不介意，然后请分享给我（
17. 追加 shell (balloon) 等的制作与配布
非常感谢！！！！！
18. 将 master shell (balloon) 修改后制成追加 shell (balloon) 配布
拒绝。
19. 将对话内容的一部份在网络上转载
附上出处和配布页地址，可以。
20. 将对话全文在网络上转载
拒绝。
21. 截图转载
如果能附上配布页地址就更好了。
22. 作为 (论坛头像、摄影截图等的) 素材使用
拒绝。
23. 直接链接到人格的下载页面
不可以，请链接到人格配布页。
24. 将公开中的人格在网络上进行再配布
绝对不行。
25. 将配布终了的人格在网络上进行再配布
绝对不行。
26. 将内容改变后在网络上进行再配布（只限 24 和 25 中某一条 OK 的人）
绝对不行。
27. 将配布终了的人格安装文件用邮件进行传播
绝对不行。
28. 将配布终了的人格安装文件从 InternetArchive 等地下载（不许可流通的人除外）
绝对不行。
29. 将安装文件收录于同人志
请和我联系。
30. 将安装文件收录于商业志
请和我联系。
